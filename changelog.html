AIUB Discobot Changelog

This page contains the latest announcements and the bot development changelogs from the AIUB Discobot developer Mushfiqur Rahman Abir. Changelog

Announces

v3.0.0

Published on: 22 March, 2024

Major Release - Version 3.0

Major release cycle - V3.0.0 is now live. The update includes API changes, new features, and bug fixes. Also note that this update is mostly for developers as it changes backend technologies a lot.

Changelog:
  • Welcome Message: A welcome message is now added when the bot is added to the server.
  • Documentation Enrichment: The documentation of the bot has been improved
  • Error Notifications: Email notifications are now sent for errors that occur while running the bot. So devs can now take quick actions.
  • Logging System: Database logging systems have been implemented.
  • Terminal Output Colors: Colors have been added to the terminal output.
  • Slash Command Update: The slash command system has been updated.
  • CLA Introduction: Few new CLA - Command Line Argument has been introduced.
  • Remaining Time Display: For developers, the remaining time is now shown in the terminal.
  • Notice Fetching: Notice fetching is now done using the AIUB-API. The fetching method has drastically been changed in this version so hope to get more prominent and fast notice fetches as the fetching is now done using external API.
  • Configuration System: A configuration system has been added
  • Dynamic Presence: The bot's discord presence is now dynamic so it will show updates in realtime if a server joins or leaves or not
  • Automatic Channel Creation: The bot is now able to create the aiub-notice channel automatically if not present in the server.
  • COGS Architecture: The AIUB-DISCOBOT source code now follows the COGS system
  • Strong Exception Handling: Exception handling has been improved in this version.
  • Overall Output Messages Updates: The overall messages/responses output format has been updated quite a bit in some comman ds
Special Notes:
  • The discobot notice fetching now fully depends on an external project which is AIUB-API. So in order to keep the bot working the AIUB-API projects needs to be running successfully

v2.0.0

Major Release - Version 2.0

This is a major release of AIUB Discobot. The update includes major architecture changes along with new features & bug fixes.

Changelog:
  • The long-awaited slash commands are now available. You can just issue / in messagebox and AIUB Discobot will show available commands which you can issue now.
  • Discobot framework changed to a more newer version.
  • Depreciation of the old prefix-based commands. So !COMMANDS will not work anymore from v2.
  • Addition of new AIUB SDD link to /website command.
  • Architectural changes according to the new discord.py V2 changes.
  • Reformatted some outputs of the commands.
  • Bugfix for the /help command and the help subcommands.
Known Issues:
  • The server count does not update dynamically.
  • The auto notice fetching is posting late for some unknown reason. And sometimes also misses a notice. I will try to fix this ASAP in the next release.

Also, a dedicated landing page for the bot is available now: AIUB Discobot Website.
